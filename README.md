# occultation-details
Gathering all information relevant to observed occultations including stellar details and asteroidal details. Meant to integrate with Google Sheets.

## Goal
To develop an agentic AI solution to the time-intensive process of searching for the details of stars and asteroids relevant for the analysis of stellar angular diameters and asteroid diameters.

## Method
Using Kiro and python to create an effective AI agent that can accomplish these tasks.
